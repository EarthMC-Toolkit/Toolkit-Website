EarthMC Stats - Usage Guide

Overview

EarthMC Stats is by far the most popular and mature of the tools in this toolkit. Quick responses and a wide range of commands with info neatly displayed in Discord embeds, give players an intuitive way to get useful info, even when the server goes down!

Invite the bot here with the following minimum permissions:
READ | SEND | ADD REACTIONS

Prefixes

Aurora: a/ or slash commands.

Nova: n/


Since slash commands only support the / prefix, this means only the latest map (currently Aurora), will be supported.

List of Commands

Regular Slash Description Arguments
town [name] town lookup [name] Get information on the specified town. N/A
town list town list Displays a holistic list of towns. online, residents, chunks, name
town activity [name] town list Displays a list of residents in the specified town, with their last online date(s). N/A
