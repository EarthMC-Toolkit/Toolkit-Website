API Documentation

The EarthMC API is served using Heroku and Express and provides plenty of data such as townless, nations, alliances and more with most endpoints being cached to decrease response times. The repository is open source and can be found here.

Quick Start

For this example, we will use JavaScript combined with the Node Package Manager to install a fetch package that will allow us to send GET requests.

Note: To stop abuse, there is a rate limit of 10 requests per 5 seconds.
1. Install a fetch package in your chosen language.
$ npm install node-fetch

2. Require/import the package.
const fetch = require('node-fetch')

3. Send a get request to the specified route, and wait for the result.
var towns = await fetch('https://earthmc-api.herokuapp.com/api/v1/aurora/towns').catch(() => {})
if (!towns) // Handle error

towns.forEach(t => {
   // Your code here
})
